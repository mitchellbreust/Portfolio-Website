Investment Appreciation Visualiser Web App

HOW IT WORKS: The user inputs in a form; an investment period, apreciation rate, initial investment, and investment contriubtion amount. After this, the user is tacked to another webpage where the result graph is shown. There is also an option to go back to the original webpage to make a new graph

TECHNOLOGY: The web application is created with react on the front end and express.js on the backend. Also, the graph is created with python's matplot lib library. I decided to use react as it gave me an easy way to change the state of the result page to be customised spesifcally for a user. Furthermore, express.js allowed me to easily accept the form data from the input page and send it to my python file (where to graph is created). Express.js can then post the graph after it has been created to the result page.
